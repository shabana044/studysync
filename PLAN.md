# StudySync Plan

## Project Goal
StudySync is a student planner PWA for managing study life in one place.

## Main Features

### 1. Dashboard
- Today's study tasks
- Upcoming exams
- Attendance summary
- Pending assignments

### 2. Subjects
- Add subjects
- Track attendance per subject
- View attendance percentage

### 3. Exams
- Add exam name, subject, date, and notes
- Show upcoming exams

### 4. Assignments
- Add assignment title, subject, due date
- Mark assignment as completed

### 5. Daily Study Tasks
- Add tasks for today
- Mark tasks as done

### 6. Reminders
- Basic reminder list
- Later: browser notifications

### 7. Offline Support
- Save data locally first
- App should work without internet

## Tech Stack
- React
- TypeScript
- Vite
- PWA
- Local storage / IndexedDB
- Charts
- GitHub
- Deployment
